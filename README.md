# RAG-application


## Requirements

1) Python 3.8 or later

2) Create a new environment
```bash
$ python -m venv env
```
3) Activate the environment

```bash
$ env/Scripts/Activate
```

4) Install required packages:
```bash
pip install -r requirements.txt
```
5) Setup environment variables
```bash 
$ cp .env.example .env
